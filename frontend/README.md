# Frontend - Accounting Software

Next.js 15 application with TypeScript and Tailwind CSS.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Features

- ✅ Next.js 15 App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ ShadCN UI components
- ✅ React Query for data fetching
- ✅ Axios API client
- ✅ Form validation
- ✅ Responsive design

## Pages

- `/` - Home (redirects to login/dashboard)
- `/login` - Login page
- `/dashboard` - Main dashboard
- `/dashboard/accounts` - Chart of Accounts
- `/dashboard/invoices` - Invoices
- `/dashboard/journal` - Journal Entries
- `/dashboard/reports/*` - Financial Reports
